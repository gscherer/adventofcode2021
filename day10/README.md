# Day 10

## Setup 

Install python 3.9.10 using pyenv

`pyenv install 3.9.10`

Create a virtualenv for day10

`pyenv virtualenv 3.9.10 day10`

Use that virtualenv for this project

`pyenv local day10`

Install dependencies

`pip install -r requirements.txt`
